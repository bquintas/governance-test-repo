# My Good Demo

A compliant demo for testing governance structure checks.

## Overview

This demo validates that the structure check workflow passes when all required files and sections are present.

## Prerequisites

- AWS CDK v2
- Python 3.9+
- Node.js 18+

## Deployment

```bash
./deploy-all.sh
```

Or on Windows:

```powershell
.\deploy-all.ps1
```
